Project 02 — Source 2 / s&box

UntitledGame II

Planned

Our second project — built on s&box and the Source 2 engine. Taking everything learned from building from scratch and applying it inside a professional-grade modern engine.

About the Game

After Protocol, we'll have a deep understanding of how game systems work at the lowest level. Untitled Game II is where we take that knowledge and work inside s&box on the Source 2 engine — one of the most capable modern engines available.

The goal here isn't to use an engine as a crutch. It's to leverage professional tools with the clarity of someone who built those tools from scratch themselves. The jump from raw C++ to a full engine is massive in terms of capability — and we intend to use every bit of it.

This project is still in early vision. Details on genre, mechanics, and setting will be revealed as Protocol progresses. What we know is that it will be ambitious, finished, and worth your time.

Development Goals
GOAL / 01
Engine Mastery
Learn Source 2 and s&box deeply — not just how to use it, but how it works and why it was designed the way it was.
GOAL / 02
Larger Scope
With engine support, push scope further — more complex worlds, better visuals, richer systems than Protocol allows.
GOAL / 03
Multiplayer Ready
s&box is built for multiplayer. This project will explore networked gameplay in ways raw C++ makes extremely difficult.

Ship Ready
A polished, complete product suitable for public release — not just a tech demo or learning exercise.
