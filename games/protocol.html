Protocol

In Development

Immersive, atmospheric storytelling where player agency and survival mechanics combine to create a visceral and high-stakes experience of being a single and vulnerable entity within an indifferent universe.

About the Game

Protocol is the first game from Wireframe Studios, and it exists for a specific reason: to build a complete game from absolute first principles. No engine abstractions. No pre-built physics systems. Every loop, every collision, every frame rendered by hand.

The goal isn't to reinvent the wheel for its own sake — it's to understand the wheel deeply enough that we could build a better one. This project is as much about mastering C++ and game architecture as it is about shipping a finished product.

The result is a game with an unusual level of intentionality behind every mechanic. When you play Protocol, you're playing something built by a developer who understands exactly why every system works the way it does.

Core Systems
SYS / 01
Custom Renderer
Rendering pipeline written from scratch — no engine abstraction between code and pixels.
SYS / 02
Physics Engine
Hand-rolled collision detection and response. Every bounce and slide calculated explicitly.
SYS / 03
Entity System
Custom entity-component architecture designed for performance and clarity.
SYS / 04
Input Handling
Low-latency input system built directly on platform APIs for precise control.
SYS / 05
Audio Engine
Lightweight audio mixer supporting positional sound and layered effects.
SYS / 06
Memory Manager
Manual memory management with custom allocators — no garbage collection crutches.
